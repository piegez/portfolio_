Linguagens

Tecnologias e Linguagens de Programação

Desenvolvimento Web

O desenvolvimento web envolve a criação e manutenção de websites e aplicações web. Abrange tanto a parte frontal (front-end), que é a interface com a qual os usuários interagem, quanto a parte de trás (back-end), que lida com a lógica de servidor e banco de dados. Como desenvolvedora júnior, estou comprometida em aprender e aplicar as melhores práticas para criar experiências web dinâmicas e responsivas.

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para descrever a apresentação de documentos HTML. Com o CSS, posso controlar a aparência visual de um site, incluindo layouts, cores e fontes. Estou sempre em busca de aprimorar minhas habilidades em CSS para criar designs atraentes e intuitivos.

React

React é uma biblioteca JavaScript para a construção de interfaces de usuário. Ele permite criar componentes reutilizáveis, tornando o desenvolvimento mais eficiente e a manutenção mais simples. Como desenvolvedora júnior, utilizo o React para desenvolver aplicações web interativas, com foco na performance e na experiência do usuário.

JavaScript

JavaScript é uma linguagem de programação fundamental para o desenvolvimento web, permitindo a criação de conteúdo dinâmico e interativo. Com o JavaScript, posso adicionar funcionalidades a sites, manipular o DOM e comunicar-me com servidores. Minha paixão por aprender e aplicar JavaScript me ajuda a construir aplicações robustas e envolventes.

Python

Python é uma linguagem de programação versátil e de fácil leitura, amplamente utilizada em diversos domínios, como desenvolvimento web, ciência de dados e automação. Estou entusiasmada em explorar suas aplicações e usar o Python para resolver problemas complexos de forma eficiente e clara.

Django

Django é um framework web em Python que facilita o desenvolvimento de aplicações robustas e escaláveis. Ele segue o padrão de arquitetura MTV (Model-Template-View), permitindo a criação rápida de aplicações web seguras e eficientes. Como desenvolvedora júnior, estou ansiosa para utilizar o Django para construir aplicações web completas e funcionais.
